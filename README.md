# Python Linear Approximation

### Description

Linear approximation algorithm. Allows you to approximate the values of power and exponential functions over a given range.

### Requirements:

Python libraries

- Numpy
- Matplotlib
- Sympy

### How can i get it?

In command line:

    pip install -r requirements.txt
